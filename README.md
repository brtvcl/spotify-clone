## Install & Build
Prerequisites
- Install Node.js
- Clone the repo

📦 After cloning the repo install dependecies with 
```sh 
npm i
```

📡  To run development server use dev script
```sh
npm run dev
```

🔧 To build use build script
```sh
npm run build
```
